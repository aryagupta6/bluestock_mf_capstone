# Dashboard Directory

This directory contains resources and documentation for building the interactive BI dashboards.

## Documentation

- **`Dashboard_Build_Guide.md`**:
  - Detailed, step-by-step instructions on setting up SQLite ODBC drivers to connect Power BI/Tableau to `db/bluestock_mf.db` or importing processed CSV files directly.
  - Outlines schema dimensions, data relationships, and the structural metrics required for all 4 dashboard pages:
    - **Page 1:** Industry Overview (Total AUM, SIP Inflows, Folio counts, AMC market share).
    - **Page 2:** Fund Performance (Scatter plots of risk/returns, scorecard rankings, NAV vs. Benchmark timeseries comparison).
    - **Page 3:** Investor Analytics (Age/Gender segmentations, income analysis, geographical distribution maps).
    - **Page 4:** Advanced Analytics (Cohort retention models, rolling Sharpe comparisons, sector HHI, and SIP default risk alerts).
