# Reports Directory

This folder contains the figures generated by the analytical scripts and the final delivery assets compiled for presentation to management.

## Deliverables

- **`figures/`**:
  - `benchmark_chart.png`: Growth chart comparing top funds vs Nifty 50 & 100 indices (from Day 4).
  - `rolling_sharpe_chart.png`: Time-varying risk-adjusted return trend of the top 5 funds.
  - `sector_hhi_chart.png`: Equity mutual funds grouped by HHI sector concentration levels.
- **`Presentation.pptx`**:
  - A clean 12-slide presentation deck detailing project scope, relational schema design, cleaning rules, scorecard mechanics, risk findings, cohort models, and final strategic recommendations.
- **`Final_Report.pdf`**:
  - A comprehensive, multi-page data analyst document report detailing methodology, relational design, scorecard ranking tables, tail-risk metrics, and operational advisory briefs.
