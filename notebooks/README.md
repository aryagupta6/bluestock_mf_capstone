# Jupyter Notebooks Directory

This folder contains the interactive research and visual walkthrough notebooks documenting each analytical milestone.

## Notebook Descriptions

1. **`EDA_Findings.ipynb`**:
   - Focuses on initial data exploration.
   - Highlights asset class distributions, fee spreads (expense ratios), and correlations between fund attributes.
2. **`Performance_Analytics.ipynb`**:
   - Focuses on Day 4 deliverables.
   - Computes daily returns, CAGR (1y/3y/5y), Sharpe and Sortino ratios, and maximum drawdown periods.
   - Conducts OLS regressions against the Nifty 100 benchmark index to extract Alpha and Beta.
   - Houses the composite weighted scorecard model and charts normalized top-performing funds.
3. **`05_advanced_analytics.ipynb`**:
   - Focuses on Day 6 deliverables.
   - Evaluates tail risk (Value at Risk and Conditional VaR), rolling 90-day Sharpe trends, equity sector HHI concentration levels, and investor acquisition cohort behaviors.
